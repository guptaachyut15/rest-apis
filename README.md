# rest-apis
Implemented GET, POST, PATCH, and DELETE APIs which perform the operations on a MongoDB collection USERS

## Setup

```bash
npm install
```
make .env file and store:

###PORT = "..."

###CONNECTION_STRING = "..."

###LOGGER_PATH = "..."

## Usage
```bash
node run index.js
```
